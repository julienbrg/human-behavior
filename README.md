# Human Behavior

A cross-chain permissionless registry that enables privacy-preserving human verification using zkSNARK proofs and
ERC-5564 stealth addresses. Users holding a Human Passport SBT on a home chain can link their stealth meta-address, then
prove ownership of derived stealth addresses on any chain without revealing their private keys.

- [Human Passport](https://passport.human.tech/)
- [w3pk SDK](https://github.com/w3hc/w3pk)

## Install

```bash
bun i
```

## Build

```bash
forge build
```

## Test

```bash
forge test
```
